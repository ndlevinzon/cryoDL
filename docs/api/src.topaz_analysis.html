Analyze cross-validation results from Topaz training.

This function loads cross-validation results, calculates performance metrics, generates plots, and provides recommendations for optimal hyperparameters.

Parameters
cv_dir:strDirectory containing cross-validation results
n_values:list[int]List of N values used in cross-validation
k_folds:intNumber of folds used in cross-validation
output_dir:str | NoneDirectory to save analysis results (defaults to cv_dir)
save_plots:boolWhether to save generated plots
show_plots:boolWhether to display plots interactively
Returns
Dictionary containing analysis results including
  • cv_results: Full cross-validation results DataFrame
  • cv_results_mean: Mean results across folds
  • cv_results_epoch: Best epoch results for each N
  • best_n: Recommended N value
  • best_epoch: Recommended number of epochs
  • plots: Dictionary of generated plots
