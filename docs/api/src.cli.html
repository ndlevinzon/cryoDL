Main entry point for interactive CLI.

Parses command-line arguments and starts the interactive cryoDL shell. Handles keyboard interrupts and other exceptions gracefully.

Command-line Arguments:
--log-file: Specify custom log file path (default: cryodl.log)

Example

python cli.py python cli.py --log-file my_session.log
